# Investigator Toolboard

**Offline-first browser app for visual investigations, OSINT, evidence boards & presentations.**

Drag & drop images/videos/audio/PDFs, connect nodes, group into frames, drop waypoints, and present with animated camera moves + spotlight focus.

100% offline – no server, no account, data saved via IndexedDB.

## Try it now
Just double-click `index.html` (or open in any browser).


## Features
- Infinite canvas
- Media embeds (images, video, audio, PDF viewer, URL iframes)
- Customizable edges with labels & curves
- Frames & layers
- Waypoint-based presentations with smooth transitions
- Undo/redo, minimap, snap grid

## How to use
1. Download or clone this repo
2. Open `index.html` in Chrome/Firefox/Edge
3. Press **?** for shortcuts
4. Ctrl+S to save board
